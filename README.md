# wellcheck-landing

WellCheck marketing website. Deployed via Firebase Hosting from the `main` branch on GitHub.

## Deploying Changes

```bash
# Stage specific files
git add index.html styles.css

# Or stage all changed files
git add -A

# Commit
git commit -m "Your message here"

# Push to GitHub (triggers Firebase deploy)
git push
```

## Common File Locations

- `index.html` — home page
- `styles.css` — shared styles
- `Images/` — AVIF screenshots and icons
- `privacy/index.html` — privacy policy
- `terms/index.html` — terms of service
- `consent/index.html` — SMS consent documentation
